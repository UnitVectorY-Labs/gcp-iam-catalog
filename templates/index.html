Welcome to gcp-iam-catalog

gcp-iam-catalog is an independent tool designed to simplify the exploration of Google Cloud Platform (GCP) Identity and Access Management (IAM) predefined roles and permissions. This tool is not affiliated with or endorsed by Google.

Why gcp-iam-catalog?

Comparing different predefined IAM roles in GCP can be complex due to the vast number of roles and permissions. gcp-iam-catalog simplifies this process by offering an intuitive interface to explore and compare IAM roles and their associated permissions.

  • Role-to-Permission Mapping: Easily view the permissions included in each IAM role across all GCP services. (Google documentation found at Understanding Roles).
  • Permission-to-Role Mapping: Identify all roles that grant a specific permission. (Google documentation found at Permissions Reference).
  • Dynamic Comparison: Select two roles or permissions to see what they have in common and what is unique to each.
  • Updated Daily: Roles and permissions automatically updated using GCP IAM API once a day.
